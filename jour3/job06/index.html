Quelles précautions devez-vous prendre pour éviter d'affecter d'autres éléments ou styles lors de la gestion des événements ?

Il faut utiliser des sélecteur css précis pour définir les règles css, que ce soit dans un fichier css ou js. On peut également utiliser la méthode stopPropagation() d'un event pour éviter que les événements ne se propagent aux enfants
