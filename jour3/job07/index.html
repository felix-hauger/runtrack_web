Comment prévenir les attaques par script intersites (XSS) lors de l'affichage de données stockées ?

Avec JavaScript on peut utiliser innerHtml qui va insérer du html, et html5 empêche les scripts de s'exécuter quand ils sont insérer avec innerHtml. Il faut cependant également le faire côté back car les scripts JavaScript côté client peuvent être altérés.
